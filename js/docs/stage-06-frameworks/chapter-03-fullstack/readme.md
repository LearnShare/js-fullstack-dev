# 6.3 全栈框架概览

## 章节概述

全栈框架提供了前后端一体的开发方案，简化了全栈应用开发。本章介绍常用的全栈框架，包括 Next.js、Nuxt.js、Remix 等，帮助你了解全栈框架的特点和使用场景。

## 学习目标

完成本章学习后，你将能够：

- 了解 Next.js 的特点和使用场景
- 了解 Nuxt.js 的特点和使用场景
- 了解 Remix 的特点和使用场景
- 了解全栈框架的选择建议

## 章节结构

本章包含以下小节：

1. **Next.js 简介**：Next.js 的核心特性和使用场景
2. **Nuxt.js 简介**：Nuxt.js 的核心特性和使用场景
3. **Remix 简介**：Remix 的核心特性和使用场景
4. **其他全栈框架**：其他全栈框架的简介

## 学习建议

1. **理解概念**：重点理解全栈框架的核心概念和特点
2. **对比学习**：对比不同框架的优缺点
3. **实践应用**：在实际项目中应用全栈框架
4. **持续学习**：全栈框架在快速发展，需要持续学习

## 预计学习时间

- **总学习时间**：约 4-6 小时
- **每日学习时间**：1-2 小时
- **预计完成时间**：2-3 天
