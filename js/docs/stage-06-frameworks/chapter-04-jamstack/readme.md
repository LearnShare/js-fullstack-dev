# 6.4 JAMstack 架构

## 章节概述

JAMstack（JavaScript、APIs、Markup）是一种现代化的 Web 开发架构，强调预渲染、解耦和性能。本章介绍 JAMstack 的概念、原理和实现方式。

## 学习目标

完成本章学习后，你将能够：

- 理解 JAMstack 的概念和原理
- 了解静态站点生成器的使用
- 了解 Headless CMS 的作用
- 了解 CDN 和边缘计算在 JAMstack 中的应用

## 章节结构

本章包含以下小节：

1. **JAMstack 概念与原理**：JAMstack 的核心概念和架构原理
2. **静态站点生成器**：常用的静态站点生成器及其使用
3. **Headless CMS**：Headless CMS 的概念和使用
4. **CDN 与边缘计算**：CDN 和边缘计算在 JAMstack 中的作用

## 学习建议

1. **理解概念**：重点理解 JAMstack 的核心概念和优势
2. **实践应用**：使用静态站点生成器创建项目
3. **架构思考**：理解 JAMstack 架构的设计思路
4. **性能优化**：理解 JAMstack 的性能优势

## 预计学习时间

- **总学习时间**：约 4-6 小时
- **每日学习时间**：1-2 小时
- **预计完成时间**：2-3 天
