## 核心概念

- **单线程模型**：JavaScript 是单线程执行的，同一时间只能执行一个任务
- **事件循环**：JavaScript 的执行机制，负责管理调用栈和任务队列
- **调用栈**：存储函数调用信息的数据结构
- **任务队列**：存储待执行任务的数据结构
- **宏任务**：setTimeout、setInterval、I/O 操作等
- **微任务**：Promise.then、queueMicrotask 等

## 学习建议

1. **理解原理**：事件循环是异步编程的基础，需要深入理解其工作原理
2. **画图辅助**：通过画图帮助理解调用栈和任务队列的交互
3. **多实践**：通过代码示例理解执行顺序
4. **调试工具**：使用浏览器调试工具观察调用栈
