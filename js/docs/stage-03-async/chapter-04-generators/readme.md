# 3.4 生成器与迭代器

## 目标

- 理解迭代器协议和可迭代对象
- 掌握生成器函数的创建和使用
- 理解生成器的暂停和恢复机制
- 掌握异步生成器的使用
- 了解生成器的实际应用场景

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释和完整示例：

1. **[生成器与迭代器概述](section-01-overview.md)**：迭代器协议、可迭代对象和生成器的基本概念

2. **[迭代器协议](section-02-iterators.md)**：迭代器协议的定义、实现和使用

## 核心概念

- **迭代器**：提供迭代机制的对象
- **可迭代对象**：实现了迭代器协议的对象
- **生成器函数**：使用 function* 定义的函数
- **yield**：暂停生成器函数执行的关键字
- **异步生成器**：使用 async function* 定义的异步生成器

## 学习建议

1. **理解协议**：深入理解迭代器协议的工作原理
2. **多实践**：通过大量练习掌握生成器的使用
3. **理解机制**：理解生成器的暂停和恢复机制
4. **实际应用**：了解生成器在实际项目中的应用
