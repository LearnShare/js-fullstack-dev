# 2.11 错误与异常处理

## 目标

- 理解 JavaScript 的错误类型
- 掌握 try...catch...finally 的使用
- 能够创建自定义错误

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明和完整示例：

1. **[错误类型](section-01-error-types.md)**：Error、TypeError、ReferenceError 等

2. **[try...catch...finally](section-02-try-catch.md)**：异常捕获、错误处理、finally 块

3. **[自定义错误](section-03-custom-errors.md)**：创建自定义错误类、错误处理最佳实践

## 核心概念

- **错误类型**：不同类型的错误
- **异常处理**：try...catch...finally
- **自定义错误**：创建特定类型的错误

## 学习建议

1. **理解概念**：重点理解错误类型和处理机制
2. **实践练习**：编写错误处理代码
3. **最佳实践**：学习错误处理的最佳实践

## 完成本章后

- 能够识别不同类型的错误
- 能够使用 try...catch 处理异常
- 能够创建自定义错误
