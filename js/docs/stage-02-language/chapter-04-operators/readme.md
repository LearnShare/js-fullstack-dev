# 2.4 运算符与表达式

## 目标

- 掌握各种运算符的使用方法
- 理解运算符的优先级和结合性
- 学会使用现代运算符（??、?.、??=）
- 理解表达式和运算符的区别

## 章节内容

本章分为五个独立小节，每节提供详细的概念解释、语法说明和完整示例：

1. **[算术运算符](section-01-arithmetic.md)**：+、-、*、/、%、**、++、--

2. **[比较运算符（=== vs ==）](section-02-comparison.md)**：==、===、!=、!==、>、<、>=、<=

3. **[逻辑运算符（&&、||、!）](section-03-logical.md)**：&&、||、!、逻辑运算的短路特性

4. **[现代运算符（??、?.、??=）](section-04-modern-operators.md)**：空值合并运算符、可选链运算符、逻辑赋值运算符

5. **[运算符优先级](section-05-precedence.md)**：运算符优先级表、结合性、括号的使用

## 核心概念

- **运算符**：执行特定操作的符号
- **表达式**：由运算符和操作数组成的代码单元
- **优先级**：运算符的执行顺序
- **结合性**：相同优先级运算符的执行方向

## 学习建议

1. **理解概念**：重点理解运算符的行为和优先级
2. **实践练习**：编写代码验证每个运算符的行为
3. **注意陷阱**：注意类型转换和优先级陷阱
4. **使用现代运算符**：掌握 ??、?. 等现代运算符

## 完成本章后

- 能够正确使用各种运算符
- 理解运算符的优先级和结合性
- 能够使用现代运算符简化代码
- 理解表达式和运算符的区别
