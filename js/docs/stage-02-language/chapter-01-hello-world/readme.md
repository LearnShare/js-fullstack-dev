# 2.1 第一个 JavaScript 程序

## 目标

- 编写并运行第一个 JavaScript 程序
- 了解 JavaScript 文件的基本结构
- 掌握语句和注释的用法
- 了解不同环境下的执行方式

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、语法说明和完整示例：

1. **[Hello World 程序](section-01-hello-world.md)**：编写第一个 JavaScript 程序，了解基本语法

2. **[文件结构与编码](section-02-file-structure.md)**：JavaScript 文件的结构、编码规范、文件组织

3. **[语句与注释](section-03-statements-comments.md)**：语句的规则、注释的用法、代码风格

4. **[执行方式（浏览器、Node.js、REPL）](section-04-execution.md)**：在不同环境中执行 JavaScript 代码的方法

## 核心概念

- **JavaScript 程序**：由语句组成的代码集合
- **语句**：执行特定操作的代码单元
- **注释**：用于说明代码的文本，不参与执行
- **执行环境**：运行 JavaScript 代码的环境

## 学习建议

1. **动手实践**：每学习一个概念，立即编写代码验证
2. **多环境尝试**：在浏览器、Node.js 和 REPL 中都运行代码
3. **理解语法**：重点理解语句的语法规则，而非死记硬背
4. **养成习惯**：从一开始就养成良好的代码风格和注释习惯

## 完成本章后

- 能够编写和运行简单的 JavaScript 程序
- 了解 JavaScript 文件的基本结构
- 掌握语句和注释的用法
- 能够在不同环境中执行 JavaScript 代码
