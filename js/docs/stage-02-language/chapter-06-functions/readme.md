# 2.6 函数

## 目标

- 掌握函数的声明和表达式
- 理解箭头函数的特点和使用
- 掌握函数参数（默认参数、剩余参数）
- 理解作用域和闭包的概念
- 了解高阶函数的使用

## 章节内容

本章分为五个独立小节，每节提供详细的概念解释、语法说明和完整示例：

1. **[函数声明与表达式](section-01-declarations.md)**：函数声明、函数表达式、函数提升

2. **[箭头函数](section-02-arrow-functions.md)**：箭头函数语法、this 绑定、使用场景

3. **[参数（默认参数、剩余参数）](section-03-parameters.md)**：默认参数、剩余参数、参数解构

4. **[作用域与闭包](section-04-scope-closures.md)**：作用域链、闭包概念、闭包应用

5. **[高阶函数](section-05-higher-order.md)**：函数作为参数、函数作为返回值、函数式编程基础

## 核心概念

- **函数声明**：使用 function 关键字声明函数
- **函数表达式**：将函数赋值给变量
- **箭头函数**：ES6 引入的简洁函数语法
- **闭包**：函数能够访问外部作用域的特性
- **高阶函数**：接受函数作为参数或返回函数的函数

## 学习建议

1. **理解概念**：重点理解函数的作用域和闭包
2. **实践练习**：编写各种类型的函数
3. **注意 this**：理解箭头函数和普通函数的 this 绑定差异
4. **使用高阶函数**：掌握函数式编程的基础

- 理解箭头函数的特点和使用场景
- 掌握函数参数的高级用法

- 能够使用高阶函数
