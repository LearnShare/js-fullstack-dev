# 2.9 Set 与 Map

## 目标

- 掌握 Set 数据结构的使用
- 掌握 Map 数据结构的使用
- 理解 WeakSet 和 WeakMap 的特点

## 章节内容

## 核心概念

- **Set**：唯一值的集合
- **Map**：键值对的集合
- **WeakSet/WeakMap**：弱引用的集合
