# 2.10 面向对象与原型

## 目标

- 理解 JavaScript 的原型链机制
- 掌握 ES6 Class 语法
- 理解继承和 super 的使用
- 理解 this 关键字的行为
- 掌握静态方法和属性

## 章节内容

本章分为五个独立小节，每节提供详细的概念解释、语法说明和完整示例：

1. **[原型链机制](section-01-prototype-chain.md)**：原型、原型链、属性查找

2. **[ES6 Class 语法](section-02-class.md)**：类声明、构造函数、方法定义

3. **[继承与 super](section-03-inheritance.md)**：类继承、super 关键字、方法重写

4. **[this 关键字](section-04-this.md)**：this 的绑定规则、箭头函数的 this

5. **[静态方法与属性](section-05-static.md)**：静态方法、静态属性、使用场景

## 核心概念

- **原型链**：JavaScript 的继承机制
- **Class**：ES6 的类语法
- **this**：函数执行上下文
- **继承**：类之间的继承关系

## 学习建议

1. **理解概念**：重点理解原型链和 this 绑定
2. **实践练习**：编写类和继承的代码
3. **对比学习**：对比原型链和 Class 语法
4. **理解 this**：理解 this 的不同绑定规则

## 完成本章后

- 能够理解和使用原型链
- 能够使用 Class 语法定义类
- 能够实现类继承
- 能够理解 this 的绑定规则
- 能够使用静态方法和属性
