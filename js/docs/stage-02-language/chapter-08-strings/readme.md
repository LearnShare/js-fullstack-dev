# 2.8 字符串操作

## 目标

- 掌握字符串的创建和模板字符串
- 掌握常用字符串方法
- 理解正则表达式的基础使用

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明和完整示例：

1. **[字符串创建与模板字符串](section-01-creation.md)**：字符串字面量、模板字符串、多行字符串

2. **[字符串方法](section-02-methods.md)**：常用字符串方法、字符串操作

3. **[正则表达式基础](section-03-regex.md)**：正则表达式创建、匹配、替换

## 核心概念

- **字符串**：不可变的字符序列
- **模板字符串**：ES6 引入的字符串插值
- **正则表达式**：模式匹配工具

## 学习建议

1. **理解概念**：重点理解字符串的不可变性
2. **实践练习**：编写代码操作字符串
3. **掌握方法**：熟练掌握常用字符串方法
4. **学习正则**：掌握正则表达式的基础使用

## 完成本章后

- 能够创建和使用字符串
- 能够使用模板字符串
- 能够使用字符串方法
- 能够使用正则表达式进行模式匹配
