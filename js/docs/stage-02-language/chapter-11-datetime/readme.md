- 理解 Date 对象的基本概念和用法

## 核心概念

- **Date 对象**：JavaScript 中表示日期和时间的对象
- **时间戳**：从 1970-01-01 00:00:00 UTC 开始的毫秒数
- **时区**：不同地区的时间差异
- **UTC**：协调世界时，标准时间参考
