# 2.3 数据类型

## 目标

- 理解 JavaScript 的数据类型系统
- 掌握原始类型（Primitives）和引用类型（Objects）的区别
- 学会使用类型检测方法（typeof、instanceof）
- 理解类型转换的机制（隐式与显式）

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、语法说明和完整示例：

1. **[原始类型（Primitives）](section-01-primitives.md)**：number、string、boolean、null、undefined、symbol、bigint

2. **[引用类型（Objects）](section-02-objects.md)**：对象、数组、函数、日期、正则表达式等

3. **[类型检测（typeof、instanceof）](section-03-type-detection.md)**：typeof 运算符、instanceof 运算符、类型检测的最佳实践

4. **[类型转换（隐式与显式）](section-04-type-conversion.md)**：隐式类型转换、显式类型转换、类型转换规则

## 核心概念

- **原始类型**：不可变的值类型
- **引用类型**：可变的对象类型
- **类型检测**：确定变量类型的方法
- **类型转换**：将值从一种类型转换为另一种类型

## 学习建议

1. **理解区别**：重点理解原始类型和引用类型的区别
2. **实践练习**：编写代码验证类型检测和转换
3. **注意陷阱**：注意类型转换的陷阱和边界情况
4. **使用工具**：使用 typeof 和 instanceof 进行类型检测

## 完成本章后

- 能够识别和使用所有原始类型
- 理解引用类型的特点
- 能够正确使用类型检测方法
- 理解类型转换的机制和规则
