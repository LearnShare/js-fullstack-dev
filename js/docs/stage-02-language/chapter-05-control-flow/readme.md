# 2.5 控制结构

## 目标

- 掌握条件语句的使用（if、else、switch）
- 掌握循环结构的使用（for、while、do...while）
- 掌握迭代器循环（for...of、for...in）
- 理解跳转语句的作用（break、continue、return）

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、语法说明和完整示例：

1. **[条件语句（if、else、switch）](section-01-conditionals.md)**：if、else if、else、switch、三元运算符

2. **[循环结构（for、while、do...while）](section-02-loops.md)**：for、while、do...while、嵌套循环

3. **[迭代器循环（for...of、for...in）](section-03-iterators.md)**：for...of、for...in、区别和使用场景

4. **[跳转语句（break、continue、return）](section-04-jump-statements.md)**：break、continue、return、标签语句

## 核心概念

- **条件语句**：根据条件执行不同的代码
- **循环结构**：重复执行代码块
- **迭代器**：遍历数据结构
- **跳转语句**：控制程序执行流程

## 学习建议

1. **理解概念**：重点理解各种控制结构的使用场景
2. **实践练习**：编写代码练习各种控制结构
3. **注意陷阱**：注意循环中的常见陷阱
4. **选择合适结构**：根据场景选择合适的控制结构

## 完成本章后

- 能够使用条件语句控制程序流程
- 能够使用循环结构重复执行代码
- 能够使用迭代器遍历数据结构
- 能够使用跳转语句控制执行流程
