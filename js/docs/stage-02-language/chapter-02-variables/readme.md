# 2.2 变量与常量

## 目标

- 掌握 `let`、`const` 和 `var` 的区别
- 理解变量声明、初始化和赋值
- 理解作用域的概念（全局、函数、块级）
- 理解暂时性死区（TDZ）的概念
- 掌握常量的使用和引用类型的特性

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、语法说明和完整示例：

1. **[变量声明（let、const）](section-01-declarations.md)**：`let` 和 `const` 的用法、变量命名规则、变量声明的最佳实践

2. **[var 的问题与暂时性死区（TDZ）](section-02-var-tdz.md)**：`var` 的问题、变量提升、暂时性死区的概念和影响

3. **[常量与引用类型](section-03-constants.md)**：`const` 的特性、常量与引用类型的关系、不可变性的概念

4. **[作用域（全局、函数、块级）](section-04-scope.md)**：作用域的概念、全局作用域、函数作用域、块级作用域、作用域链

## 核心概念

- **变量声明**：使用 `let`、`const` 或 `var` 声明变量
- **变量提升**：`var` 声明的变量会被提升到作用域顶部
- **暂时性死区**：`let` 和 `const` 在声明前无法访问
- **作用域**：变量可访问的范围
- **常量**：使用 `const` 声明的不可重新赋值的变量

## 学习建议

1. **理解概念**：重点理解作用域、变量提升、暂时性死区等概念
2. **对比学习**：对比 `let`、`const` 和 `var` 的差异
3. **实践练习**：编写代码验证每个概念
4. **避免 `var`**：在现代开发中，优先使用 `let` 和 `const`

## 完成本章后

- 能够正确使用 `let` 和 `const` 声明变量
- 理解 `var` 的问题并避免使用
- 理解作用域的概念和作用域链
- 理解暂时性死区的概念
- 掌握常量的使用和引用类型的特性
