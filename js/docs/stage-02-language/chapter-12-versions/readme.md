# 2.12 JavaScript 版本新特性

## 目标

- 了解 ES2015 及以后版本的新特性
- 理解各版本的重要更新
- 保持对语言发展的了解

## 章节内容

本章简要介绍 JavaScript 各版本的重要新特性，帮助学习者了解语言的发展趋势。

2. **实践使用**：尝试使用新特性
3. **关注兼容性**：注意浏览器和 Node.js 的兼容性
