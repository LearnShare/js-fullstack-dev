# 2.7 对象与数组

## 目标

- 掌握对象的创建和操作
- 掌握数组的基础操作和高级方法
- 理解解构赋值的使用
- 理解展开运算符的使用

## 章节内容

本章分为五个独立小节，每节提供详细的概念解释、语法说明和完整示例：

1. **[对象创建与操作](section-01-objects.md)**：对象字面量、属性访问、方法定义

2. **[数组基础与操作](section-02-arrays.md)**：数组创建、索引访问、长度属性

3. **[数组方法（map、filter、reduce）](section-03-array-methods.md)**：map、filter、reduce、forEach 等高级方法

4. **[解构赋值](section-04-destructuring.md)**：对象解构、数组解构、参数解构

5. **[展开运算符](section-05-spread.md)**：数组展开、对象展开、函数参数展开

## 核心概念

- **对象**：键值对的集合
- **数组**：有序的元素集合
- **解构赋值**：从对象或数组中提取值
- **展开运算符**：展开数组或对象

## 学习建议

1. **理解概念**：重点理解对象和数组的特性
2. **实践练习**：编写代码操作对象和数组
3. **掌握方法**：熟练掌握数组的高级方法
4. **使用现代语法**：掌握解构和展开运算符

## 完成本章后

- 能够创建和操作对象
- 能够创建和操作数组
- 能够使用数组的高级方法
- 能够使用解构赋值和展开运算符
