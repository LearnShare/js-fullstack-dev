# 1.2 开发工具链

## 目标

- 了解现代 JavaScript 开发工具链的组成
- 掌握代码编辑器和 IDE 的配置
- 学习包管理器的使用
- 了解构建工具的作用和配置
- 掌握代码质量工具的使用
- 学习调试工具和技巧

## 章节内容

本章分为六个独立小节，每节提供详细的概念解释、安装步骤和完整示例：

1. **[开发工具链概述](section-01-overview.md)**：开发工具链的基本概念、组成部分、工具选择原则

2. **[代码编辑器与 IDE 配置](section-02-editor-ide.md)**：VS Code、WebStorm 等编辑器的配置、插件推荐、调试配置

3. **[包管理器（npm、pnpm、yarn）](section-03-package-managers.md)**：npm、pnpm、yarn 的使用、对比、最佳实践

4. **[构建工具（Vite、Webpack、Turbopack）](section-04-build-tools.md)**：Vite、Webpack、Turbopack 的配置和使用、对比分析

5. **[代码质量工具（ESLint、Prettier）](section-05-code-quality.md)**：ESLint 和 Prettier 的配置、规则定制、集成使用

6. **[调试工具与技巧](section-06-debugging.md)**：浏览器调试、Node.js 调试、VS Code 调试配置、调试技巧

## 核心概念

- **开发工具链**：开发过程中使用的工具集合
- **包管理器**：管理项目依赖的工具
- **构建工具**：编译、打包、优化代码的工具
- **代码质量工具**：检查代码质量和格式的工具
- **调试工具**：帮助定位和修复问题的工具

## 学习建议

1. **按顺序学习**：按顺序学习六个小节，理解完整的开发工具链

2. **重点掌握**：
   - 代码编辑器的配置和使用
   - 包管理器的基本操作
   - 构建工具的基本配置
   - 代码质量工具的配置
   - 调试工具的使用

3. **实践练习**：
   - 配置代码编辑器
   - 创建项目并安装依赖
   - 配置构建工具
   - 设置代码质量工具
   - 练习调试技巧

## 完成本章后

- 能够配置和使用现代开发工具链
- 掌握包管理器的使用
- 能够配置构建工具
- 能够使用代码质量工具
- 掌握调试工具和技巧
