# 1.1 JavaScript 运行环境

## 目标

- 了解 JavaScript 在不同环境中的运行方式
- 理解浏览器环境与 V8 引擎的工作原理
- 掌握 Node.js 环境的安装和配置
- 了解其他现代运行时（Deno、Bun）的特点

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、安装步骤和完整示例：

1. **[JavaScript 运行环境概述](section-01-overview.md)**：JavaScript 运行环境的基本概念、不同环境的对比、运行时的作用

2. **[浏览器环境与 V8 引擎](section-02-browser-v8.md)**：浏览器中的 JavaScript 执行、V8 引擎的工作原理、浏览器 API 的提供

3. **[Node.js 环境](section-03-nodejs.md)**：Node.js 的安装与配置、核心模块、npm 包管理器、Node.js 与浏览器的差异

4. **[其他运行时（Deno、Bun）](section-04-other-runtimes.md)**：Deno 和 Bun 的特点、安装与使用、与 Node.js 的对比

## 核心概念

- **运行时环境**：执行 JavaScript 代码的环境
- **JavaScript 引擎**：解析和执行 JavaScript 代码的引擎
- **V8 引擎**：Google 开发的高性能 JavaScript 引擎
- **Node.js**：基于 V8 的服务器端 JavaScript 运行时
- **Deno**：现代、安全的 JavaScript/TypeScript 运行时
- **Bun**：高性能的 JavaScript 运行时

## 学习建议

1. **按顺序学习**：按顺序学习四个小节，理解 JavaScript 在不同环境中的运行方式

2. **重点掌握**：
   - 浏览器环境和 Node.js 环境的区别
   - V8 引擎的基本工作原理
   - Node.js 的安装和基本使用
   - 不同运行时的特点和适用场景

3. **实践练习**：
   - 在浏览器中运行 JavaScript 代码
   - 安装和配置 Node.js
   - 尝试使用 Deno 或 Bun
   - 对比不同运行时的性能

## 完成本章后

- 能够理解 JavaScript 在不同环境中的运行方式
- 了解浏览器环境和 Node.js 环境的特点和差异
- 掌握 Node.js 的安装和基本使用
- 了解 Deno 和 Bun 等现代运行时的特点
