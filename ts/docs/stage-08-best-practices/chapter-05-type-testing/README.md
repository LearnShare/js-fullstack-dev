# 8.5 类型测试

## 章节概述

类型测试可以验证类型定义的正确性。本章介绍类型测试的方法，包括 tsd 类型测试、dtslint 类型测试、类型测试最佳实践。

## 学习目标

完成本章学习后，你将能够：

- 使用 tsd 进行类型测试
- 使用 dtslint 进行类型测试
- 遵循类型测试最佳实践

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础类型系统
- 声明文件的基础
- 测试的基础
