# 8.3 性能优化

## 章节概述

性能优化是 TypeScript 项目的重要方面。本章介绍 TypeScript 性能优化的方法，包括编译性能优化、类型检查性能、项目引用。

## 学习目标

完成本章学习后，你将能够：

- 优化 TypeScript 编译性能
- 优化类型检查性能
- 使用项目引用

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础配置
- 模块系统的基础
- 项目结构的基础

## 章节结构

本章包含以下小节：

1. **编译性能优化**：优化 TypeScript 编译性能
2. **类型检查性能**：优化类型检查性能
3. **项目引用（Project References）**：使用项目引用优化性能

## 学习建议

1. **理解性能**：重点理解性能优化的方法
