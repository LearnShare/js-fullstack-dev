# 8.7 TypeScript 在全栈开发中的应用

## 章节概述

TypeScript 在全栈开发中发挥着重要作用，特别是在前后端类型共享、API 类型定义等方面。本章介绍如何在全栈项目中使用 TypeScript，实现类型安全的全栈开发。

## 学习目标

完成本章学习后，你将能够：

- 理解 TypeScript 在全栈开发中的作用
- 实现前后端类型共享
- 定义 API 类型
- 构建类型安全的全栈应用

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础类型系统
- 接口和类型别名
- 泛型基础
- 模块系统

## 章节结构

本章包含以下小节：

1. **全栈开发概述**：TypeScript 在全栈开发中的作用
2. **前后端类型共享**：实现前后端类型共享的方法
3. **API 类型定义**：定义 API 请求和响应的类型
4. **类型安全的全栈开发**：构建类型安全的全栈应用

1. **实践为主**：通过实际项目理解全栈开发
