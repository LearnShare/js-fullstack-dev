# 8.6 与构建工具集成

## 章节概述

TypeScript 与构建工具的集成是生产环境的重要部分。本章介绍 TypeScript 与构建工具的集成，包括 Vite + TypeScript、Webpack + TypeScript、Turbopack + TypeScript。

## 学习目标

完成本章学习后，你将能够：

- 配置 Vite + TypeScript
- 配置 Webpack + TypeScript
- 配置 Turbopack + TypeScript

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础配置
- 构建工具的基础
- 模块系统的基础

## 章节结构

本章包含以下小节：

1. **Vite + TypeScript**：配置 Vite + TypeScript
2. **Webpack + TypeScript**：配置 Webpack + TypeScript
3. **Turbopack + TypeScript**：配置 Turbopack + TypeScript

## 学习建议

1. **理解集成**：重点理解构建工具的集成方式
2. **实践为主**：通过实践掌握集成配置
3. **选择工具**：根据项目选择合适的构建工具
4. **实际应用**：理解集成在实际项目中的应用

- **每日学习时间**：1-2 小时
