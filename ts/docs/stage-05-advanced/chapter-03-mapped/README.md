# 5.3 映射类型（Mapped Types）

## 章节概述

映射类型允许基于旧类型创建新类型。本章介绍映射类型的定义、使用和高级特性。

## 学习目标

完成本章学习后，你将能够：

- 理解映射类型的概念和作用
- 定义和使用映射类型
- 使用键重映射
- 使用模板字面量类型

## 前置知识

学习本章前，需要掌握：

- TypeScript 泛型基础
- 条件类型的使用
- keyof 操作符的使用

## 章节结构

本章包含以下小节：

1. **映射类型概述**：映射类型的概念、作用和语法
2. **映射类型基础**：映射类型的基本用法
3. **键重映射（Key Remapping）**：键重映射的使用
4. **模板字面量类型**：模板字面量类型的使用

## 学习建议

1. **理解概念**：重点理解映射类型的概念和作用
2. **实践为主**：通过实践掌握映射类型的用法
3. **键重映射**：理解键重映射的使用
4. **模板字面量**：掌握模板字面量类型的使用

- **每日学习时间**：1-2 小时
- **预计完成时间**：2-3 天
