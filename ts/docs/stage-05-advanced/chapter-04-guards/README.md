# 5.4 类型守卫（Type Guards）

## 章节概述

类型守卫用于在运行时检查类型，并在类型检查器中收窄类型。本章介绍类型守卫的定义和使用。

## 学习目标

完成本章学习后，你将能够：

- 理解类型守卫的概念和作用
- 使用 typeof 和 instanceof
- 创建自定义类型守卫
- 使用可辨识联合

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础类型系统
- 联合类型的使用
- 条件类型的基础

## 章节结构

本章包含以下小节：

1. **类型守卫概述**：类型守卫的概念、作用和分类
2. **typeof 与 instanceof**：typeof 和 instanceof 的使用
3. **用户自定义类型守卫**：创建自定义类型守卫
4. **可辨识联合（Discriminated Unions）**：可辨识联合的使用

## 学习建议

1. **理解概念**：重点理解类型守卫的概念和作用
2. **实践为主**：通过实践掌握类型守卫的用法
3. **类型收窄**：理解类型收窄的工作原理
4. **实际应用**：理解类型守卫在实际项目中的应用

- **每日学习时间**：1-2 小时
- **预计完成时间**：2 天
