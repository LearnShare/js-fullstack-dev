# 5.6 类型兼容性

## 章节概述

类型兼容性是 TypeScript 类型系统的基础，决定了哪些类型可以相互赋值。本章介绍类型兼容性的规则和原理。

## 学习目标

完成本章学习后，你将能够：

- 理解类型兼容性的概念和规则
- 理解结构化类型系统
- 理解协变与逆变
- 理解类型兼容性的实际应用

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础类型系统
- 接口和类型别名的使用
- 函数类型的使用

## 章节结构

本章包含以下小节：

1. **类型兼容性概述**：类型兼容性的概念、规则和作用
2. **结构化类型系统深入**：结构化类型系统的原理
3. **协变与逆变**：协变和逆变的概念和应用
4. **类型兼容性规则**：类型兼容性的详细规则

## 学习建议

1. **理解概念**：重点理解类型兼容性的概念和规则
2. **实践为主**：通过实践理解类型兼容性
