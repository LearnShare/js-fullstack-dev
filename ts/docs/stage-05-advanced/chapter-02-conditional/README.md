# 5.2 条件类型（Conditional Types）

## 章节概述

条件类型允许根据条件选择类型。本章介绍条件类型的定义、使用和高级特性。

## 学习目标

完成本章学习后，你将能够：

- 理解条件类型的概念和作用
- 定义和使用条件类型
- 理解分布式条件类型
- 使用 infer 关键字进行类型推断

## 前置知识

学习本章前，需要掌握：

- TypeScript 泛型基础
- 工具类型的使用
- 映射类型的基础

## 章节结构

本章包含以下小节：

1. **条件类型概述**：条件类型的概念、作用和语法
2. **条件类型基础**：条件类型的基本用法
3. **分布式条件类型**：分布式条件类型的行为
4. **infer 关键字**：使用 infer 进行类型推断

## 学习建议

1. **理解概念**：重点理解条件类型的概念和作用
2. **实践为主**：通过实践掌握条件类型的用法
3. **分布式理解**：理解分布式条件类型的行为
4. **类型推断**：掌握 infer 关键字的使用

- **每日学习时间**：1-2 小时
