# 3.3 现代类特性

## 章节概述

TypeScript 提供了许多现代类特性，包括参数属性、私有字段、装饰器等。本章介绍这些现代特性的定义和使用。

## 学习目标

完成本章学习后，你将能够：

- 理解现代类特性的概念和作用
- 使用参数属性简化代码
- 使用私有字段实现真正的私有
- 理解和使用装饰器
- 创建自定义装饰器

## 前置知识

学习本章前，需要掌握：

- TypeScript 类的基础
- 继承和多态的使用
- 访问修饰符的使用

## 章节结构

本章包含以下小节：

1. **现代类特性概述**：现代类特性的概念和作用
2. **参数属性**：参数属性的定义和使用
3. **私有字段**：私有字段的定义和使用
4. **装饰器**：装饰器的基本概念和使用
5. **装饰器深入**：装饰器的高级用法
6. **装饰器元数据**：装饰器元数据的使用
7. **自定义装饰器**：创建自定义装饰器
8. **装饰器组合**：装饰器的组合使用

## 学习建议

1. **理解概念**：重点理解现代类特性的概念和作用
2. **实践为主**：通过实践掌握各种现代特性的用法
3. **装饰器学习**：装饰器是重点内容，需要深入学习
4. **实际应用**：理解现代特性在实际项目中的应用
