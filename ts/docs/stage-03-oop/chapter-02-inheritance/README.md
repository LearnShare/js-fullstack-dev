# 3.2 继承与多态

## 章节概述

继承是面向对象编程的核心特性，允许子类继承父类的属性和方法。本章介绍 TypeScript 类的继承、多态、抽象类和接口实现。

## 学习目标

完成本章学习后，你将能够：

- 理解继承的概念和作用
- 实现类继承
- 使用 super 关键字
- 理解抽象类和抽象方法
- 实现接口

## 前置知识

学习本章前，需要掌握：

- TypeScript 类的基础
- 访问修饰符的使用
- 接口的基本概念

## 章节结构

本章包含以下小节：

1. **继承与多态概述**：继承和多态的概念、作用和优势
2. **类继承（extends）**：使用 extends 实现类继承
3. **super 关键字**：super 关键字的使用
4. **抽象类与抽象方法**：抽象类和抽象方法的定义和使用
5. **接口实现（implements）**：类实现接口

## 学习建议

1. **理解概念**：重点理解继承和多态的概念
2. **实践为主**：通过实践掌握继承的各种用法
3. **super 使用**：理解 super 关键字的作用和使用场景
4. **抽象类**：理解抽象类和接口的区别
