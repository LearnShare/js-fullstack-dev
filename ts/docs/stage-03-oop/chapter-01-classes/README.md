# 3.1 类的基础

## 章节概述

类是 TypeScript 面向对象编程的基础。本章介绍类的基础知识，包括类声明、实例化、构造函数、访问修饰符、只读属性和静态成员。

## 学习目标

完成本章学习后，你将能够：

- 理解类的概念和作用
- 声明和实例化类
- 使用构造函数
- 理解和使用访问修饰符
- 使用只读属性和静态成员

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础类型系统
- 接口和类型别名的使用
- JavaScript ES6 Class 语法基础

## 章节结构

本章包含以下小节：

1. **类的基础概述**：类的概念、作用和优势
2. **类声明与实例化**：类的声明和实例化方法
3. **构造函数**：构造函数的使用和参数
4. **访问修饰符**：public、private、protected 的使用
5. **只读属性与静态成员**：只读属性和静态成员的使用

## 学习建议

1. **理解概念**：重点理解类的概念和面向对象编程思想
2. **实践为主**：通过实践掌握类的各种用法
3. **访问控制**：理解访问修饰符的作用和使用场景
4. **静态成员**：理解静态成员和实例成员的区别

## 预计学习时间

- **总学习时间**：约 3-4 小时
- **每日学习时间**：1-2 小时
- **预计完成时间**：2-3 天
