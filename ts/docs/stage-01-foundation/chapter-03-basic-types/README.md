# 1.2 基础类型系统

## 章节概述

TypeScript 的类型系统是 TypeScript 的核心特性。本章介绍 TypeScript 的基础类型系统，包括原始类型、数组、元组、枚举、特殊类型、类型推断、类型注解和类型断言。

## 学习目标

完成本章学习后，你将能够：

- 理解 TypeScript 的类型系统基础
- 使用原始类型（string、number、boolean）
- 使用数组和元组类型
- 使用枚举类型
- 理解特殊类型（any、unknown、void、never）
- 掌握类型推断和类型注解
- 使用类型断言

## 前置知识

学习本章前，需要掌握：

- JavaScript 基础语法
- JavaScript 数据类型
- TypeScript 安装与配置（已完成）

## 章节结构

本章包含以下小节：

1. **基础类型系统概述**：类型系统的基本概念和作用
2. **原始类型（string、number、boolean）**：原始类型的使用
3. **数组与元组**：数组类型和元组类型
4. **枚举类型**：枚举的定义和使用
5. **特殊类型（any、unknown、void、never）**：特殊类型的使用场景
6. **类型推断与类型注解**：类型推断和显式类型注解
7. **类型断言**：类型断言的使用和注意事项

## 学习建议

1. **理解概念**：理解类型系统的基本概念和作用
2. **实践为主**：通过实践掌握各种类型的用法
3. **类型安全**：理解类型安全的重要性
4. **类型推断**：理解类型推断的工作原理
