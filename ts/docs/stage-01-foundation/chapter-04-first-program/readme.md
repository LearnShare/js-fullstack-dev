# 1.3 第一个 TypeScript 程序

## 章节概述

编写第一个 TypeScript 程序是学习 TypeScript 的重要里程碑。本章通过实际动手实践，帮助学习者创建、编译和运行第一个 TypeScript 程序，体验 TypeScript 的类型检查和编译过程。

## 学习目标

完成本章学习后，你将能够：

- 创建第一个 TypeScript 程序
- 理解 TypeScript 编译过程
- 运行编译后的 JavaScript 代码
- 体验类型检查的作用

## 前置知识

学习本章前，需要掌握：

- TypeScript 安装与配置（已完成）
- 基础类型系统（已学习）
- 命令行基本操作

## 学习建议
