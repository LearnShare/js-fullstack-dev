# 1.1 TypeScript 安装与配置

## 章节概述

TypeScript 的安装和配置是学习 TypeScript 的第一步。本章介绍如何安装 TypeScript、配置 tsconfig.json、了解编译器选项、配置 IDE、使用直接执行工具以及掌握编译操作。

## 学习目标

完成本章学习后，你将能够：

- 安装 TypeScript 编译器
- 配置 tsconfig.json 文件
- 理解编译器选项的含义
- 配置 IDE 支持 TypeScript
- 使用 TypeScript 直接执行工具
- 掌握 TypeScript 编译操作

## 前置知识

学习本章前，需要掌握：

- Node.js 基础使用
- npm/yarn/pnpm 包管理器使用
- 命令行基本操作

## 章节结构

本章包含以下小节：

1. **TypeScript 安装**：全局安装和项目安装
2. **tsconfig.json 详解**：配置文件的结构和选项
3. **编译器选项**：各种编译器选项的说明
4. **IDE 配置**：VSCode、WebStorm 等 IDE 的配置
5. **TypeScript 直接执行工具**：ts-node、tsx、bun、deno 等工具的使用
6. **编译操作与实践**：tsc 命令的使用、编译模式、监视模式等

## 学习建议

1. **实践为主**：每学习一个配置项，立即实践
2. **理解选项**：理解编译器选项的作用和影响
3. **工具使用**：掌握直接执行工具的使用场景
4. **编译实践**：通过实际编译操作理解编译过程

## 预计学习时间

- **总学习时间**：约 4-6 小时
- **每日学习时间**：1-2 小时
- **预计完成时间**：2-3 天
