# 4.2 泛型约束

## 章节概述

泛型约束允许限制类型参数的范围，确保类型参数具有某些属性或方法。本章介绍泛型约束的使用，包括 extends 约束、keyof 操作符、条件类型约束。

## 学习目标

完成本章学习后，你将能够：

- 理解泛型约束的概念和作用
- 使用 extends 约束类型参数
- 使用 keyof 操作符
- 使用条件类型约束

## 前置知识

学习本章前，需要掌握：

- TypeScript 泛型基础
- 接口和类型别名的使用
- 类的基础知识

## 章节结构

本章包含以下小节：

1. **泛型约束概述**：泛型约束的概念、作用和优势
2. **extends 约束**：使用 extends 约束类型参数
3. **keyof 操作符**：keyof 操作符的使用
4. **条件类型约束**：条件类型约束的使用

## 学习建议

1. **理解概念**：重点理解泛型约束的概念和作用
2. **实践为主**：通过实践掌握各种约束的用法
3. **约束类型**：理解不同类型约束的作用
4. **实际应用**：理解约束在实际项目中的应用
