# 6.1 模块系统

## 章节概述

模块系统是 TypeScript 工程化的基础。本章介绍 TypeScript 的模块系统，包括 ES Modules、CommonJS、模块解析策略、类型导入。

## 学习目标

完成本章学习后，你将能够：

- 理解模块系统的概念和作用
- 使用 ES Modules 和 CommonJS
- 配置模块解析策略
- 使用类型导入

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础类型系统
- JavaScript 模块系统基础
- 包管理和依赖基础

## 章节结构

本章包含以下小节：

1. **模块系统概述**：模块系统的概念、作用和分类
2. **ES Modules 与 TypeScript**：ES Modules 的使用
3. **CommonJS 与 TypeScript**：CommonJS 的使用
4. **模块解析策略**：模块解析策略的配置
5. **类型导入（import type）**：类型导入的使用

## 学习建议

1. **理解概念**：重点理解模块系统的概念和作用
2. **实践为主**：通过实践掌握模块系统的使用
3. **配置理解**：理解模块解析策略的配置
4. **类型导入**：理解类型导入的作用

- **总学习时间**：约 2-3 小时
- **每日学习时间**：1-2 小时
- **预计完成时间**：2 天
