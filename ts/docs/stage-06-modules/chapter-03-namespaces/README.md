# 6.3 命名空间（Namespace）

## 章节概述

命名空间用于组织相关的类型和值。本章介绍命名空间的使用，包括基础、合并、与模块的对比。

## 学习目标

完成本章学习后，你将能够：

- 理解命名空间的概念和作用
- 使用命名空间组织代码
- 理解命名空间合并
- 理解命名空间与模块的区别

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础类型系统
- 模块系统的基础
- 声明文件的基础

## 章节结构

本章包含以下小节：

1. **命名空间基础**：命名空间的概念、定义和使用
2. **命名空间合并**：命名空间合并的规则
3. **命名空间 vs 模块**：命名空间与模块的对比
