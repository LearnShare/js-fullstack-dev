# 6.2 声明文件（.d.ts）

## 章节概述

声明文件用于为 JavaScript 代码提供类型定义。本章介绍声明文件的编写，包括基础、类型、全局声明、模块声明、命名空间、类型扩展、DefinitelyTyped、自定义声明文件、第三方库处理、最佳实践。

## 学习目标

完成本章学习后，你将能够：

- 理解声明文件的概念和作用
- 编写各种类型的声明文件
- 扩展第三方库类型
- 处理第三方库声明文件

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础类型系统
- 接口和类型别名的使用
- 模块系统的基础

## 章节结构

本章包含以下小节：

1. **声明文件基础**：声明文件的概念、作用和语法
2. **声明文件类型**：不同类型的声明文件
3. **全局声明文件**：全局声明文件的编写
4. **模块声明文件**：模块声明文件的编写
5. **declare module 与 declare global**：declare 关键字的使用
6. **declare namespace**：命名空间声明
7. **类型扩展与合并**：类型扩展和合并
8. **DefinitelyTyped (@types/)**：DefinitelyTyped 的使用
9. **编写自定义声明文件**：自定义声明文件的编写
10. **第三方库声明文件处理**：第三方库声明文件的处理
11. **声明文件最佳实践**：声明文件的最佳实践

## 学习建议

1. **理解概念**：重点理解声明文件的概念和作用
