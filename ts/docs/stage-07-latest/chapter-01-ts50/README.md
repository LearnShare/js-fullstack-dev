# 7.1 TypeScript 5.0 新特性

## 章节概述

TypeScript 5.0 是一个重大版本更新，引入了多项重要新特性。本章介绍 TypeScript 5.0 的新特性，包括 const 类型参数、装饰器元数据、其他改进。

## 学习目标

完成本章学习后，你将能够：

- 理解 const 类型参数的使用
- 理解装饰器元数据的使用
- 了解 TypeScript 5.0 的其他改进

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础类型系统
- 泛型的使用
- 装饰器的基础

## 章节结构

本章包含以下小节：
