# 2.2 类型别名（Type Alias）

## 章节概述

类型别名是 TypeScript 中用于创建类型名称的方式，可以简化复杂类型的定义。本章介绍类型别名的定义、联合类型、交叉类型、字面量类型以及与接口的区别。

## 学习目标

完成本章学习后，你将能够：

- 理解类型别名的概念和作用
- 定义和使用类型别名
- 使用联合类型和交叉类型
- 理解字面量类型
- 理解接口与类型别名的区别

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础类型系统
- 接口的基本使用
- 类型注解的使用

## 章节结构

本章包含以下小节：

1. **类型别名概述**：类型别名的概念、作用和优势
2. **类型别名基础**：类型别名的定义和基本使用
3. **联合类型与交叉类型**：联合类型和交叉类型的定义和使用
4. **字面量类型**：字面量类型的定义和使用
5. **Interface vs Type**：接口与类型别名的对比

## 学习建议

1. **理解概念**：重点理解类型别名的作用和优势

4. **应用场景**：理解类型别名在实际项目中的应用
