# 2.1 接口（Interface）

## 章节概述

接口是 TypeScript 结构化类型系统的核心，用于定义对象的形状。本章介绍接口的定义、使用、继承和合并等特性。

## 学习目标

完成本章学习后，你将能够：

- 理解接口的概念和作用
- 定义和使用接口
- 使用可选属性和只读属性
- 定义函数类型和索引签名
- 理解接口继承和合并

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础类型系统
- 对象类型的基本概念
- 类型注解的使用

## 章节结构

本章包含以下小节：

1. **接口概述**：接口的概念、作用和优势
2. **接口基础**：接口的定义和基本使用
3. **可选属性与只读属性**：可选属性和只读属性的使用
4. **函数类型与索引签名**：接口中的函数类型和索引签名
5. **接口继承与合并**：接口继承和声明合并

## 学习建议

1. **理解概念**：重点理解接口的作用和结构化类型系统
2. **实践为主**：通过实践掌握接口的各种用法
3. **对比学习**：理解接口与类型别名的区别
4. **应用场景**：理解接口在实际项目中的应用

- **每日学习时间**：1-2 小时
- **预计完成时间**：2-3 天
