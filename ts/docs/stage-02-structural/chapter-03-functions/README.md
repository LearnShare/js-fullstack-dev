# 2.3 函数类型

## 章节概述

函数类型是 TypeScript 中用于描述函数签名的类型。本章介绍函数类型的定义、可选参数、默认参数、剩余参数、函数重载和函数签名类型。

## 学习目标

完成本章学习后，你将能够：

- 理解函数类型的概念和作用
- 定义和使用函数类型
- 使用可选参数和默认参数
- 理解剩余参数和函数重载
- 掌握函数签名类型的使用

## 前置知识

学习本章前，需要掌握：

- TypeScript 基础类型系统
- 接口和类型别名的基本使用
- JavaScript 函数的基本概念

## 章节结构

本章包含以下小节：

1. **函数类型概述**：函数类型的概念、作用和优势
2. **函数类型定义**：函数类型的定义方式和使用
3. **可选参数与默认参数**：可选参数和默认参数的使用
4. **剩余参数与函数重载**：剩余参数和函数重载的使用
5. **函数签名类型**：函数签名类型的定义和使用

## 学习建议

1. **理解概念**：重点理解函数类型的作用和优势
2. **实践为主**：通过实践掌握函数类型的各种用法
3. **参数处理**：理解可选参数、默认参数和剩余参数的使用
4. **函数重载**：理解函数重载的使用场景和限制
