# 2.4 类型兼容性基础

## 章节概述

类型兼容性是 TypeScript 类型系统的核心概念之一。本章介绍 TypeScript 的结构化类型系统、类型兼容性的基本规则，帮助学习者理解 TypeScript 如何判断两个类型是否兼容。

## 学习目标

完成本章学习后，你将能够：

- 理解 TypeScript 的结构化类型系统
- 掌握类型兼容性的基本规则
- 理解为什么某些类型可以互相赋值
- 理解类型兼容性在实际开发中的应用

## 前置知识

学习本章前，需要掌握：

- 接口（Interface）基础
- 类型别名（Type Alias）基础
- 函数类型基础

## 章节结构

本章包含以下小节：

1. **类型兼容性概述**：类型兼容性的基本概念和作用
2. **结构化类型系统**：理解 TypeScript 的"鸭子类型"特性
3. **类型兼容性规则基础**：基本的类型兼容性规则

## 学习建议

1. **理解概念**：重点理解结构化类型系统的特点
2. **实践为主**：通过代码示例理解类型兼容性规则
3. **对比思考**：思考为什么 TypeScript 允许某些看似不兼容的类型赋值
