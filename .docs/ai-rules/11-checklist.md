# 自检清单（提交前必须逐项确认）

- [ ] 文件命名符合规范（小写、连字符）

- [ ] 根目录 `readme.md` 中的链接正确

- [ ] 章节编号从 X.1 开始，不使用 X.0

## Markdown 格式

- [ ] 标题前后有空行
- [ ] 代码块前后有空行
- [ ] 段落之间有空行

- [ ] 表格宽度比例正确（9 中文 ≈ 15 英文）

- [ ] 语言标识使用简写（`ts`、`js`、`bash` 等）
- [ ] 行宽控制良好（<100 字符）
- [ ] 列表格式正确
- [ ] 嵌套列表缩进一致
- [ ] 列表内代码块和表格格式正确

## 代码规范

- [ ] 所有代码使用 TypeScript（100%）
- [ ] 代码语言标签使用 `ts`（不是 `typescript`）
- [ ] 所有函数参数有类型注解

- [ ] 所有 Node.js 内置模块使用 `node:` 前缀
- [ ] 使用 ESM 语法（`import` 而非 `require`）
- [ ] 所有类型都已正确导入

- [ ] 符合 12 要素要求（见 [内容完整性要求](05-content-completeness.md)）
