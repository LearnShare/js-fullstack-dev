# 阶段一：Node.js 运行时与架构（基础）

## 1. 概述

阶段一旨在构建 Node.js 全栈开发的底层知识体系。重点涵盖 Node.js 的运行时架构、核心组件协作逻辑、工业级环境搭建以及 TypeScript 工程化初始化。通过本阶段的学习，开发者应掌握 Node.js 的异步非阻塞原理及其在多核环境下的并行计算机制。

## 3. 学习目标

- **理解原理**：掌握 V8 即时编译（JIT）与 Libuv 事件循环的物理执行过程。
- **环境就绪**：完成基于版本管理器的 Node.js 22+ 环境搭建。
- **工程化起步**：构建符合现代 ESM 规范的 TypeScript 开发底座。
