# 1.3 Node.js 环境搭建与版本管理

## 1. 概述

本章介绍如何从零构建一个生产级 Node.js 开发环境。重点探讨 Node.js 物理安装的底层逻辑、环境变量（PATH）的作用机制、版本管理器（nvm/fnm）的调度原理以及 Node.js 命令行界面（CLI）的基础操作。掌握这些内容是确保开发链条稳定、解决权限冲突及版本兼容性问题的物理前提。

| **1.3.3**| [环境逻辑验证与 CLI 基础](section-03-verification.md)           | 运行时自诊断、REPL 模式、核心 Flag 应用。        |

## 3. 学习目标

- **环境构建**：能够在 Windows/Linux/macOS 下独立完成标准化的 Node.js 开发环境配置。
- **版本调度**：掌握在多项目并行开发中，通过版本管理器实现运行时版本秒级切换。
- **命令行操作**：熟练运用 CLI 参数进行运行时状态监控与基础逻辑验证。
