# 1.1 Node.js 发展史

## 1. 概述

本章解析 Node.js 的技术起源、版本演进逻辑及其在 2025 年全栈开发生态中的定位。重点探讨 Node.js 如何通过异步非阻塞架构解决 C10K 并发难题，并详细说明 LTS 版本发布策略对企业级应用的稳定性保障作用。

## 3. 学习目标

- **历史推演**：理解为何 Node.js 选择了 V8 与异步非阻塞架构。
- **版本控制**：掌握如何根据项目需求选择最稳定的 LTS 版本。
- **生态定位**：明确 Node.js 在微服务与全栈架构中的边界与优势。
