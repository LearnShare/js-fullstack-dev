# 1.2 Node.js 运行时架构深度解析

## 1. 概述

本章深入剖析 Node.js 的内部构造与底层调度机制。重点涵盖 V8 引擎的即时编译（JIT）逻辑、Libuv 的事件循环（Event Loop）阶段、单线程异步非阻塞模型的物理实现以及 Worker Threads 并行计算机制。掌握本章内容是开发者从“代码编写者”进阶为“架构设计者”的底层分水岭。

| **1.2.2**| [V8 引擎工作流水线与内存逻辑](section-02-v8.md)                 | Ignition 解释器、TurboFan 优化器及 GC 机制。     |

## 3. 学习目标

- **底层建模**：能够抽象并描述出 Node.js 处理一个异步 I/O 请求的完整物理链路。
- **性能诊断**：掌握通过事件循环时序分析排查主线程阻塞与死锁问题的方法。
- **并行实践**：理解在 Node.js 中如何安全、高效地执行 CPU 密集型计算任务。
