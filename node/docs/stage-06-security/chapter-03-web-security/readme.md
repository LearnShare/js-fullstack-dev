# 6.3 Web 安全

## 章节概述

Web 安全是应用安全的重要组成部分，涉及多种安全威胁和防护措施。本章介绍常见的 Web 安全威胁（OWASP Top 10）和防护方法，包括 XSS、CSRF、SQL 注入等。

## 学习目标

完成本章学习后，你将能够：

- 理解常见的 Web 安全威胁
- 防护 XSS 攻击
- 防护 CSRF 攻击
- 防护 SQL 注入
- 使用 Helmet.js 设置安全头

## 前置知识

学习本章前，需要掌握：

- HTTP 协议基础
- Web 框架基础
- 数据库基础
- 前端基础

## 章节结构

本章包含以下小节：

1. **Web 安全概述**：Web 安全的基本概念和威胁
2. **OWASP Top 10（2025）**：OWASP Top 10 安全威胁
3. **XSS 防护**：跨站脚本攻击的防护
4. **CSRF 防护**：跨站请求伪造的防护
5. **SQL 注入防护**：SQL 注入攻击的防护
6. **Helmet.js 安全头**：使用 Helmet.js 设置安全头

## 学习建议

1. **理解威胁**：深入理解各种安全威胁
2. **实践为主**：通过实际项目理解防护方法
3. **多层防护**：实现多层安全防护
4. **持续更新**：关注安全漏洞和最佳实践
