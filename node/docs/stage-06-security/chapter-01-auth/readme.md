# 6.1 身份认证（Authentication）

## 章节概述

身份认证是验证用户身份的过程，是应用安全的第一道防线。本章介绍常见的身份认证方式，包括 Session + Cookie、JWT、OAuth 2.0 和 Passport.js。

## 学习目标

完成本章学习后，你将能够：

- 理解身份认证的概念和重要性
- 实现 Session + Cookie 认证
- 使用 JWT 进行身份认证
- 集成 OAuth 2.0 和 OIDC
- 使用 Passport.js 简化认证流程

## 前置知识

学习本章前，需要掌握：

- HTTP 协议基础
- Cookie 和 Session 概念
- Web 框架基础
- 数据库基础

## 章节结构

本章包含以下小节：

1. **身份认证概述**：身份认证的基本概念和方法
2. **Session + Cookie**：基于 Session 和 Cookie 的认证
3. **JWT（JSON Web Token）**：基于 JWT 的认证
4. **OAuth 2.0 与 OIDC**：OAuth 2.0 和 OIDC 集成
5. **Passport.js**：使用 Passport.js 简化认证

## 学习建议

1. **理解原理**：深入理解各种认证方式的原理
2. **实践为主**：通过实际项目理解认证的实现
3. **安全考虑**：注意认证的安全性
4. **最佳实践**：遵循认证最佳实践
