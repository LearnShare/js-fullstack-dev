# 7.8 性能测试

## 章节概述

性能测试是测试应用性能的测试，包括压力测试、负载测试等。本章介绍性能测试的概念、压力测试和负载测试，以及 k6 和 Artillery 等性能测试工具。

## 学习目标

完成本章学习后，你将能够：

- 理解性能测试的概念和重要性
- 进行压力测试和负载测试
- 使用 k6 进行性能测试
- 使用 Artillery 进行性能测试

## 前置知识

学习本章前，需要掌握：

- 测试基础
- HTTP 协议基础
- 性能概念

## 章节结构

本章包含以下小节：

1. **性能测试概述**：性能测试的基本概念和类型
2. **压力测试与负载测试**：压力测试和负载测试的实现
3. **k6 与 Artillery**：k6 和 Artillery 的使用

## 学习建议

1. **理解概念**：深入理解性能测试概念
2. **实践为主**：通过实际项目理解性能测试
3. **工具使用**：掌握性能测试工具的使用
4. **最佳实践**：遵循性能测试最佳实践

- **每日学习时间**：2-3 小时
