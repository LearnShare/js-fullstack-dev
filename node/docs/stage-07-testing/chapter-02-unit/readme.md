# 7.2 单元测试

## 章节概述

单元测试是测试金字塔的基础，是对代码最小单元的测试。本章介绍单元测试的概念、Vitest 和 Jest 的使用，以及测试覆盖率的相关内容。

## 学习目标

完成本章学习后，你将能够：

- 理解单元测试的概念和重要性
- 使用 Vitest 编写单元测试
- 使用 Jest 编写单元测试
- 理解和使用测试覆盖率

## 前置知识

学习本章前，需要掌握：

- 测试基础
- TypeScript 基础
- Node.js 基础

## 章节结构

本章包含以下小节：

1. **单元测试概述**：单元测试的基本概念和特点
2. **Vitest（现代测试框架）**：Vitest 的使用
3. **Jest（传统测试框架）**：Jest 的使用
4. **测试覆盖率**：测试覆盖率的概念和使用

## 学习建议

1. **实践为主**：通过实际项目理解单元测试
2. **工具对比**：对比 Vitest 和 Jest 的特点
3. **工具选择**：根据项目需求选择合适的工具
4. **最佳实践**：遵循单元测试最佳实践

- **每日学习时间**：2-3 小时
