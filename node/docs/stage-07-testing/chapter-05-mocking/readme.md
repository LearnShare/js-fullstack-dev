# 7.5 Mock 与 Stub

## 章节概述

Mock 和 Stub 是测试中的重要技术，用于隔离被测试代码的依赖，使测试更加独立和可控。本章介绍 Mock 和 Stub 的概念、基础使用、Sinon.js 和 MSW 的使用。

## 学习目标

完成本章学习后，你将能够：

- 理解 Mock 和 Stub 的概念和区别
- 使用 Mock 和 Stub 进行测试
- 使用 Sinon.js 进行 Mock
- 使用 MSW 进行 API Mock

## 前置知识

学习本章前，需要掌握：

- 单元测试基础
- JavaScript/TypeScript 基础

## 章节结构

本章包含以下小节：

1. **Mock 与 Stub 概述**：Mock 和 Stub 的基本概念和区别
2. **Mock 基础**：Mock 的基础使用
3. **Sinon.js**：Sinon.js 的使用
4. **MSW（Mock Service Worker）**：MSW 的使用

## 学习建议

1. **理解概念**：深入理解 Mock 和 Stub 的概念
2. **实践为主**：通过实际项目理解 Mock 的使用
3. **工具使用**：掌握 Mock 工具的使用
4. **最佳实践**：遵循 Mock 使用最佳实践
