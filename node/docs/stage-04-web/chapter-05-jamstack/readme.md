# 4.5 JAMstack 架构

## 章节概述

JAMstack（JavaScript、APIs、Markup）是一种现代 Web 开发架构，通过预渲染和 CDN 提供高性能、安全的 Web 应用。本章介绍 JAMstack 架构的概念、原理、工具和实践。

## 学习目标

完成本章学习后，你将能够：

- 理解 JAMstack 架构的概念和优势
- 掌握静态站点生成器的使用
- 了解 Headless CMS 的集成
- 理解 CDN 和边缘计算的作用
- 在实际项目中应用 JAMstack 架构

## 前置知识

学习本章前，需要掌握：

- Web 框架基础（阶段四内容）
- 前端开发基础
- API 设计基础

## 章节结构

本章包含以下小节：

1. **JAMstack 概念与原理**：JAMstack 架构的基本概念和原理
2. **静态站点生成器**：静态站点生成器的使用和工具
3. **Headless CMS**：Headless CMS 的集成和使用
4. **CDN 与边缘计算**：CDN 和边缘计算的作用和实现

## 学习建议

1. **理解架构**：深入理解 JAMstack 架构的优势和适用场景
2. **实践为主**：通过实际项目理解 JAMstack 的使用
3. **工具选择**：根据项目需求选择合适的工具
4. **性能优化**：理解 JAMstack 的性能优化策略
