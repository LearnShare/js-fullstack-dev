# 4.4 全栈框架概览

## 章节概述

全栈框架提供了前后端一体化的开发体验，简化了全栈应用的开发流程。本章介绍主流的全栈框架，包括 Next.js、Nuxt.js、Remix 等，帮助开发者理解各框架的特点和选择适合的框架。

## 学习目标

完成本章学习后，你将能够：

- 理解全栈框架的概念和优势
- 掌握 Next.js 的使用和特性
- 了解 Nuxt.js 和 Remix 的特点
- 在实际项目中正确选择和使用全栈框架

## 前置知识

学习本章前，需要掌握：

- Web 框架基础（阶段四内容）
- React 或 Vue 基础
- TypeScript 基础

## 章节结构

本章包含以下小节：

1. **Next.js 简介**：Next.js 框架的使用和特性
2. **Nuxt.js 简介**：Nuxt.js 框架的使用和特性
3. **Remix 简介**：Remix 框架的使用和特性
4. **其他全栈框架**：其他重要的全栈框架

## 学习建议

1. **对比学习**：对比不同全栈框架的优缺点和适用场景
2. **实践为主**：通过实际项目理解各框架的使用
3. **场景选择**：根据项目需求选择合适的全栈框架
4. **性能考虑**：理解各框架的性能特点

## 预计学习时间

- **总学习时间**：约 10-12 小时
- **每日学习时间**：1-2 小时
- **预计完成时间**：5-6 天
