# 5.3 ORM 框架概览

## 章节概述

ORM（Object-Relational Mapping）框架将数据库表映射为对象，简化数据库操作。本章介绍主流的 ORM 框架，包括 Prisma、Drizzle、TypeORM 和 Sequelize。

## 学习目标

完成本章学习后，你将能够：

- 理解 ORM 框架的概念和优势
- 掌握 Prisma 的使用
- 了解 Drizzle ORM 的使用
- 了解 TypeORM 和 Sequelize 的使用
- 根据项目需求选择合适的 ORM 框架

## 前置知识

学习本章前，需要掌握：

- 关系型数据库基础
- TypeScript 基础
- Node.js 基础

## 章节结构

本章包含以下小节：

1. **ORM 框架概述**：ORM 框架的基本概念和特点
2. **Prisma 简介**：Prisma ORM 的使用
3. **Drizzle ORM 简介**：Drizzle ORM 的使用
4. **TypeORM 简介**：TypeORM 的使用
5. **Sequelize 简介**：Sequelize 的使用
