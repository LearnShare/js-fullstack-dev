# 5.8 消息队列

## 章节概述

消息队列是异步处理的重要机制，用于解耦系统组件、处理异步任务、实现分布式系统通信。本章介绍消息队列的概念、RabbitMQ、Bull/BullMQ 和任务调度的使用。

## 学习目标

完成本章学习后，你将能够：

- 理解消息队列的概念和用途
- 掌握 RabbitMQ 的使用
- 使用 Bull/BullMQ 实现任务队列
- 实现任务调度功能

## 前置知识

学习本章前，需要掌握：

- 异步编程概念
- 事件驱动概念
- Node.js 基础

## 章节结构

本章包含以下小节：

1. **消息队列概述**：消息队列的基本概念和用途
2. **RabbitMQ**：RabbitMQ 消息队列的使用
3. **Bull/BullMQ（Redis 队列）**：基于 Redis 的任务队列
4. **任务调度（node-cron、agenda）**：任务调度工具的使用

## 学习建议

1. **理解概念**：深入理解消息队列的概念
