# 5.2 关系型数据库（SQL）

## 章节概述

关系型数据库是应用开发中最常用的数据库类型，使用 SQL 语言进行数据操作。本章介绍关系型数据库的概念和主流数据库的使用，包括 PostgreSQL、MySQL 和 SQLite。

## 学习目标

完成本章学习后，你将能够：

- 理解关系型数据库的概念和特点
- 掌握 PostgreSQL 的使用
- 了解 MySQL 的使用
- 了解 SQLite 的使用

## 前置知识

学习本章前，需要掌握：

- SQL 基础
- 数据库设计基础
- Node.js 基础

## 章节结构

本章包含以下小节：

1. **关系型数据库概述**：关系型数据库的基本概念和特点
2. **PostgreSQL（推荐）**：PostgreSQL 数据库的使用
3. **MySQL**：MySQL 数据库的使用
4. **SQLite**：SQLite 数据库的使用

## 学习建议

1. **实践为主**：通过实际项目理解数据库使用
2. **工具选择**：根据项目需求选择合适的数据库
3. **性能优化**：理解数据库性能优化方法
4. **最佳实践**：遵循数据库使用最佳实践

- **总学习时间**：约 8-10 小时
- **每日学习时间**：1-2 小时
- **预计完成时间**：4-5 天
