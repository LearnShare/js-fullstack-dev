# 2.6 加密与安全（crypto）

## 章节概述

加密与安全模块（crypto）是 Node.js 提供加密功能的核心模块。本章介绍如何使用 crypto 模块进行哈希计算、加密解密、数字签名等安全操作。理解加密模块的使用对于构建安全的 Node.js 应用至关重要。

## 学习目标

完成本章学习后，你将能够：

- 理解加密与安全的基本概念
- 掌握哈希算法的使用
- 掌握加密和解密操作
- 理解数字签名的原理和使用
- 在实际项目中应用加密功能

## 前置知识

学习本章前，需要掌握：

- Node.js 核心模块基础
- 安全基础知识
- 异步编程基础

## 章节结构

本章包含以下小节：

1. **加密与安全概述**：加密模块的基本概念和作用
2. **哈希算法**：哈希算法的使用和应用
3. **加密与解密**：对称和非对称加密的使用
4. **数字签名**：数字签名的原理和使用

## 学习建议

4. **实际应用**：在实际项目中应用加密功能
