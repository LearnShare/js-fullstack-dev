# 2.4 HTTP 模块

## 章节概述

HTTP 模块是 Node.js 构建 Web 服务器和客户端的基础。本章介绍如何使用 Node.js 原生的 HTTP 模块创建 HTTP 服务器和客户端，处理 HTTP 请求和响应，理解 HTTP 协议的基本概念。

## 学习目标

完成本章学习后，你将能够：

- 理解 HTTP 协议的基本概念
- 使用 HTTP 模块创建服务器
- 处理 HTTP 请求和响应
- 使用 HTTP 模块创建客户端
- 在实际项目中应用 HTTP 模块

## 前置知识

学习本章前，需要掌握：

- Node.js 核心模块基础（已学习）
- 事件系统基础
- 异步编程基础

## 章节结构

本章包含以下小节：

1. **HTTP 模块概述**：HTTP 模块的基本概念和作用
2. **创建 HTTP 服务器**：使用 HTTP 模块创建服务器
3. **处理请求与响应**：处理 HTTP 请求和响应
4. **HTTP 客户端**：使用 HTTP 模块创建客户端

## 学习建议

1. **理解协议**：理解 HTTP 协议的基本概念

3. **错误处理**：注意 HTTP 请求和响应的错误处理
4. **实际应用**：在实际项目中应用 HTTP 模块
