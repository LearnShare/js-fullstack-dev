# 2.11 子进程与集群

## 章节概述

child_process 和 cluster 模块提供了创建和管理子进程的能力。child_process 用于创建子进程执行外部命令或脚本，cluster 用于创建进程集群以充分利用多核 CPU。理解子进程和集群的使用对于并行处理、任务分发、性能优化等场景非常重要。

## 学习目标

完成本章学习后，你将能够：

- 理解子进程和集群的基本概念
- 掌握 child_process 模块的使用
- 掌握 cluster 模块的使用
- 在实际项目中应用子进程和集群

## 前置知识

学习本章前，需要掌握：

- Node.js 核心模块基础
- 进程基础知识
- 异步编程基础
