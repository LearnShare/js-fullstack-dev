# 2.2 文件系统（fs）

## 章节概述

文件系统模块（fs）是 Node.js 最常用的核心模块之一，提供了与文件系统交互的 API。本章介绍如何使用 fs 模块进行文件读写、目录操作、文件监听等操作，以及同步与异步 API 的区别和使用场景。

## 学习目标

完成本章学习后，你将能够：

- 理解文件系统模块的基本概念
- 掌握文件读写操作（同步与异步）
- 使用 Promise 版本的 fs API
- 进行目录操作和文件监听
- 在实际项目中应用文件系统操作

## 前置知识

学习本章前，需要掌握：

- Node.js 模块系统（已学习）
- 异步编程基础（Promise、async/await）
- TypeScript 基础

## 章节结构

本章包含以下小节：

1. **文件系统概述**：文件系统模块的基本概念和作用
2. **文件读写（同步与异步）**：文件读写操作的方法和区别
3. **Promise 版本（fs/promises）**：使用 Promise 版本的 fs API
4. **目录操作**：目录的创建、删除、遍历等操作
5. **文件监听（watch）**：监听文件变化的方法

## 学习建议

1. **理解区别**：重点理解同步与异步 API 的区别和使用场景
2. **实践为主**：通过代码示例理解文件系统操作
3. **错误处理**：注意文件操作的错误处理
4. **性能考虑**：理解异步操作对性能的影响
