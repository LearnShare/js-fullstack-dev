# 2.9 工具函数（util）

## 章节概述

util 模块提供了 Node.js 中常用的工具函数，包括回调函数转 Promise、类型检查、调试工具等。这些工具函数可以简化代码，提高开发效率。理解 util 模块的使用对于编写高质量的 Node.js 代码很有帮助。

## 学习目标

完成本章学习后，你将能够：

- 理解 util 模块的基本功能
- 掌握 promisify 的使用
- 掌握类型检查工具的使用
- 掌握调试工具的使用
- 在实际项目中应用 util 模块

## 前置知识

学习本章前，需要掌握：

- Node.js 核心模块基础
- 异步编程基础
- Promise 和 async/await

## 章节结构

本章包含以下小节：

1. **util 模块概述**：util 模块的基本概念和功能
2. **promisify 与回调转换**：将回调函数转换为 Promise
3. **类型检查与调试工具**：类型检查和调试相关的工具函数
4. **其他工具函数**：其他常用的工具函数
