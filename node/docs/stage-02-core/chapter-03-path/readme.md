# 2.3 路径处理（path）

## 章节概述

路径处理是文件系统操作的基础。Node.js 的 `path` 模块提供了跨平台的路径处理功能，包括路径拼接、解析、规范化等操作。理解路径处理对于编写跨平台兼容的 Node.js 应用至关重要。

## 学习目标

完成本章学习后，你将能够：

- 理解路径处理的基本概念
- 掌握路径拼接与解析方法
- 获取路径信息（文件名、扩展名、目录等）
- 处理跨平台路径兼容性
- 在实际项目中应用路径处理

## 前置知识

学习本章前，需要掌握：

- 文件系统基础（已学习）
- 模块系统基础（已学习）

## 章节结构

本章包含以下小节：

1. **路径拼接与解析**：路径拼接、解析、规范化等操作
2. **路径信息获取**：获取文件名、扩展名、目录等路径信息
3. **跨平台兼容性**：处理不同操作系统的路径差异
