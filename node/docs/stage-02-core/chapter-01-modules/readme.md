# 2.1 模块系统

## 章节概述

模块系统是 Node.js 的核心特性之一，它允许开发者将代码组织成可重用的模块。本章介绍 Node.js 的模块系统，包括 CommonJS 和 ES Modules (ESM) 两种模块系统，以及模块解析与加载机制。

## 学习目标

完成本章学习后，你将能够：

- 理解 Node.js 模块系统的基本概念
- 掌握 CommonJS 模块的使用
- 掌握 ES Modules (ESM) 的使用
- 理解模块解析与加载机制
- 在实际项目中正确选择和使用模块系统

## 前置知识

学习本章前，需要掌握：

- JavaScript 基础语法
- TypeScript 基础
- Node.js 基础（阶段一内容）

## 章节结构

本章包含以下小节：

1. **模块系统概述**：模块系统的基本概念和作用
2. **CommonJS 模块**：CommonJS 模块的语法和使用
3. **ES Modules (ESM)**：ESM 模块的语法和使用
4. **模块解析与加载**：模块解析机制和加载流程

## 学习建议

1. **理解概念**：重点理解两种模块系统的区别和适用场景
2. **实践为主**：通过代码示例理解模块系统的使用
3. **对比学习**：对比 CommonJS 与 ESM 的差异
4. **实际应用**：在实际项目中应用模块系统
