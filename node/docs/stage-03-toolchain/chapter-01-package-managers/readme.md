# 3.1 包管理器

## 章节概述

包管理器是 Node.js 生态系统的核心工具，用于管理项目依赖、安装包、处理版本冲突等。本章介绍 npm、pnpm、yarn 等主流包管理器的使用，以及 package.json 的配置。理解包管理器的使用对于 Node.js 开发至关重要。

## 学习目标

完成本章学习后，你将能够：

- 理解包管理器的基本概念和作用
- 掌握 npm 的基本使用
- 掌握 pnpm 的使用和优势
- 掌握 yarn 的使用
- 理解 package.json 的配置

## 前置知识

学习本章前，需要掌握：

- Node.js 基础
- 命令行基础
- 项目结构基础

## 章节结构

本章包含以下小节：

1. **包管理器概述**：包管理器的基本概念和作用
2. **npm 基础**：npm 的基本使用
3. **pnpm（现代包管理器）**：pnpm 的使用和优势
4. **yarn（经典包管理器）**：yarn 的使用
5. **package.json 详解**：package.json 的配置详解

## 学习建议

1. **实践为主**：通过实际操作理解包管理器的使用
2. **对比学习**：对比不同包管理器的优缺点
3. **理解原理**：理解包管理的工作原理
4. **实际应用**：在实际项目中应用包管理器
