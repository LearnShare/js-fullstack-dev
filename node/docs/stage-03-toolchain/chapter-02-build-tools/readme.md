# 3.2 构建工具

## 章节概述

构建工具用于编译、打包、优化代码，将源代码转换为可执行的代码。Node.js 生态系统中有多个构建工具，包括 Vite（现代快速构建工具）、Turbopack（Next.js 的构建工具）、Webpack（传统构建工具）等。理解构建工具的使用对于现代 Node.js 开发至关重要。

## 学习目标

完成本章学习后，你将能够：

- 理解构建工具的基本概念和作用
- 掌握 Vite 的使用
- 了解 Turbopack 的特点
- 了解 Webpack 的基础使用
- 根据项目需求选择合适的构建工具

## 前置知识

学习本章前，需要掌握：

- Node.js 基础
- TypeScript 基础
- 包管理器基础

## 章节结构

本章包含以下小节：

1. **构建工具概述**：构建工具的基本概念和作用
2. **Vite（现代构建工具）**：Vite 的使用和配置
3. **Turbopack（Next.js 构建工具）**：Turbopack 的特点和使用
4. **Webpack（传统构建工具）**：Webpack 的基础使用
