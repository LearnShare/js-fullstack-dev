# 3.4 Monorepo 管理

## 章节概述

Monorepo 是一种将多个相关项目放在同一个代码仓库中的代码组织方式。Monorepo 可以提高代码复用、简化依赖管理、统一构建和测试流程。本章介绍 Turborepo、Nx、pnpm Workspaces 等 Monorepo 管理工具的使用。

## 学习目标

完成本章学习后，你将能够：

- 理解 Monorepo 的概念和优势
- 掌握 Turborepo 的使用
- 了解 Nx 的特点
- 掌握 pnpm Workspaces 的使用
- 在实际项目中应用 Monorepo

## 前置知识

学习本章前，需要掌握：

- Node.js 基础
- 包管理器基础
- 项目结构基础
