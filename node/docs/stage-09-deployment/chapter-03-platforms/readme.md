# 9.3 部署平台

## 章节概述

部署平台提供了应用运行的环境，包括 PaaS、Serverless、IaaS 等不同类型的平台。本章介绍各种部署平台的特点、使用场景和部署方法。

## 学习目标

完成本章学习后，你将能够：

- 理解不同部署平台的特点
- 部署到 PaaS 平台
- 部署到 Serverless 平台
- 部署到 IaaS 平台

## 前置知识

学习本章前，需要掌握：

- 应用开发基础
- 网络基础
- 云服务基础

## 章节结构

本章包含以下小节：

1. **部署平台概述**：不同部署平台的特点和选择
2. **PaaS（Vercel、Railway、Render）**：PaaS 平台的部署
3. **Serverless（AWS Lambda、Cloudflare Workers）**：Serverless 平台的部署
4. **IaaS（AWS EC2、DigitalOcean）**：IaaS 平台的部署

## 学习建议

1. **理解差异**：理解不同平台的特点
2. **实践为主**：通过实际项目理解部署
3. **工具使用**：掌握部署工具的使用
4. **最佳实践**：遵循部署最佳实践

## 预计学习时间

- **总学习时间**：约 10-12 小时
- **每日学习时间**：2-3 小时
- **预计完成时间**：4-5 天
